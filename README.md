# NBA Trade Analyzer

Backend scoring engine for evaluating NBA trades across cap, basketball fit, and future-vs-goal dimensions.

Status: early scaffolding. See [`docs/brainstorms/2026-05-29-trade-scoring-backend-requirements.md`](docs/brainstorms/2026-05-29-trade-scoring-backend-requirements.md) for the working design.

## Stack
- Node.js
- Plain JavaScript (no TypeScript)
- Vitest

## Commands
```bash
npm install
npm test
npm run test:watch
```
